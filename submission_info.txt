📖 Project Summary
------------------------------------------------------------------------
A modular MLOps-structured web dashboard featuring:
1. Traditional NLP Preprocessing: Word/sentence tokenization, POS tag distributions, lexical diversity, and Flesch Reading Ease calculations.
2. Custom Heuristics Fallback: Heuristic sentiment scores, regex entity extraction, and TF-IDF sentence-score summarization when running locally.
3. Advanced LLM Integration: Google Gemini prompt engineering interface for full summaries, emotional scales (Joy, Trust, Fear, Anger, Sadness), structured entity lists, and history-aware conversational Q&A.
4. Glassmorphic UI: Sleek dark-mode interface with dynamic SVG/CSS bar charts, sentiment circular gauges, and inline chat consoles.

🛠️ Local Setup & Run instructions (For Evaluators)
------------------------------------------------------------------------
1. Extract the submission ZIP file.
2. Open terminal in the directory and run:
   python -m venv venv
   # Activate:
   # (Windows) .\venv\Scripts\activate
   # (macOS/Linux) source venv/bin/activate
3. Install package:
   pip install -r requirements.txt
   pip install -e .
4. Start FastAPI server:
   python -m uvicorn app:app --reload --host 127.0.0.1 --port 8000
5. Open browser at: http://127.0.0.1:8000
